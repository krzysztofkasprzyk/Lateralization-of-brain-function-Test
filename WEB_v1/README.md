# WEB v1

Przegladarkowa wersja testu lateralizacji przygotowana pod Google Chrome.

Najwazniejsze:

- dziala bez instalowania bibliotek Python
- dziala jako statyczne pliki HTML / CSS / JS
- uruchamia sie bezposrednio w Chrome
- obsluguje trening, 256 prob glownych, pauze, pelny ekran i eksport CSV

Uruchamianie:

- Windows: `APP/URUCHOM_W_CHROME.bat`
- macOS: `APP/URUCHOM_W_CHROME.command`

Pliki:

- `APP/` - gotowa aplikacja webowa
- `PAKIET-WEB-CHROME-V1.zip` - spakowana paczka do przekazania
- `TESTY/test_web_chrome.ps1` - test headless Chrome pod Windows
