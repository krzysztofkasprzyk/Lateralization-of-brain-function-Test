WERSJA WEB V1

To jest przegladarkowa wersja testu lateralizacji przygotowana pod Google Chrome.

Najwazniejsze:

- dziala bez instalowania bibliotek Python
- dziala jako statyczne pliki HTML/CSS/JS
- uruchamia sie bezposrednio w Chrome
- obsluguje trening, 256 prob glownych, pauze, licznik prob i eksport CSV

Jak uruchomic:

Windows:
- otworz folder APP
- kliknij URUCHOM_W_CHROME.bat

macOS:
- otworz folder APP
- kliknij URUCHOM_W_CHROME.command

Pliki:

- APP/index.html
  glowna aplikacja

- APP/app.js
  logika eksperymentu

- APP/styles.css
  wyglad

Uwaga:

- w Chrome mozna wybrac plik CSV przed startem
- jesli plik nie zostanie wybrany, CSV bedzie do pobrania po zakonczeniu badania
