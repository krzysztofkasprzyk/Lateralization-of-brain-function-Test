# Test lateralizacji

Repozytorium zawiera gotowy instalator programu oraz dokumentację użytkownika.

## Zawartość

- `INSTALLER/TestLateralizacjiSetup.exe` - instalator programu dla Windows
- `DOKUMENTACJA/Instrukcja_Test_Lateralizacji.docx` - instrukcja użytkownika
- `DOKUMENTACJA/README.md` - krótki opis uruchomienia i podstawowej obsługi

## Szybki start

1. Pobierz `INSTALLER/TestLateralizacjiSetup.exe`.
2. Uruchom instalator i wybierz folder instalacji.
3. Po instalacji uruchom program ze skrótu w menu Start lub z folderu aplikacji.
4. Szczegółowe kroki są opisane w dokumentacji w folderze `DOKUMENTACJA`.
